## 品牌叙事（闭环自持 · v3 视觉化）
**里海 = 被大地环抱的海：一百多条河注入，没有一个出海口——Nothing flows out。** HERO 用**真实里海地理轮廓 SVG**（北宽、阿普歇伦收腰、南盆浑圆）+ 五条河流只进不出的载入编排动画直接把这层隐喻画出来；logo 换为里海封闭轮廓 ₿ 标记。设计系统仍承自胡侃比特（瓷器教义/弦纹/底款印章 ≋+₿），玄海鎏金釉色。
