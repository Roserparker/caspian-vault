## 兼容性（2026-07-08 大陆主流设备优化轮）
- 断点体系按 iPad 竖屏 768 重整（960/760/720/520）；村账本 auto-fit 网格；导航窄屏单行横滑。
- 弃用老内核雷点：`overflow-x:clip`→hidden、`inset`→四边全写、`aspect-ratio`→固定高、flex gap 有兜底、`-webkit-sticky` 前缀。
- 微信/国内浏览器：`format-detection` 防数字变电话、`renderer=webkit` 双核声明、og 标签齐全。
- 性能：波浪动画隔帧+页面隐藏休眠、四处轮询合并单心跳、reduced-motion 全链降级。
