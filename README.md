## 品牌叙事
**里海 = 地球上最大的内流海：河流只注入，海水永不外流。** 公司把资产负债表建成一座里海——只进不出的金库。名字自带五行（水），签名视觉=右缘「水位刻度」随滚动蓄满。

## 页面结构（窗户纸教学法：靶→破→喻→数据→金句）
1. **英雄区**：内流海动画（金线河流注入夜色海盆，海面微澜）
2. **先理解钱**：账本世界观 + 尺子思维（互动：沙发×缩水的尺，2.00→3.71→换硬尺）
3. **为什么是比特币**：能量本位（$60/克、93万度电 count-up）+ 21M 供应阶梯曲线 + 法币轮回时间线
4. **区块链三分钟**：互动①九本村账 8:1 否决假账；互动②篡改旧区块→链条级联断裂
5. **我们做什么**：四根柱（托管/传承/身份合规/税务清晰）+ 每股含币量唯一仪表 + 飞轮与五条刹车 + 三条红线卡
6. **诚实条款**：归零=全损顶置 + 四次 -75%+ 回撤柱 + 免责
7. **路线图**：教育(已上线)→Delaware 起步壳(●你在这里)→创始记录期→极小范围同行者→亚洲架构
8. 页脚：熟人引荐制、导流胡侃比特、完整免责
