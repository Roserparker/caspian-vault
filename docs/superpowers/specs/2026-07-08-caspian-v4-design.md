# Caspian Vault 官网 v4 设计规格 · 「钢青精密 / 深海图录」

> 2026-07-08 定案。经三套配方对比板 + 逐项问答确认。
> 本文档只管**怎么好看、怎么读**；不改变 v3 的内容主张与合规措辞（后者已标「待律师终审」处照原样保留）。

## 0 · 背景：v3 的五条病根（站长反馈）

1. HERO 河流箭头先于海出现，观感差；
2. 海形不符真实地理；
3. 站内存在简笔画式示意图（如「其一·独立」的椭圆+箭头），**简笔画为大忌**；
4. 整体风格与胡侃比特过近，且偏严肃、英文过多、字体不考究；
5. Slogan 只有英文。

v4 的答案：一套全新美学「钢青精密」，以**高雅**为执行标准——冷可以，糙不行。

## 1 · 美学系统

### 1.1 气质定位

**深海图录**：苏富比图录的排版纪律 × 深水的细腻光感。不是审计底稿、不是工程机房；是一册被灯光照着的深海珍图。与胡侃比特（暖纸瓷器、人文书房）彻底分开：那边暖、这边冷；那边纸、这边水；那边手工、这边精工。

### 1.2 色彩令牌

| 令牌 | 值 | 用途 |
|---|---|---|
| `--steel-black` | `#070B0E` | 全站底色 |
| `--deep-teal` | `#0E1A20` | 面板/卡片底 |
| `--steel` | `#31586B` | 细线、结构、等深线 |
| `--titan` | `#9FC6D4` | 强调青、图形主线 |
| `--silver` | `#D9DEE2` | 正文字 |
| `--mist` | `#7D8894` | 弱文字、图注 |
| `--btc` | `#E8813A` | **全站唯一暖色** |
| `--btc-dim` | `#8A5A34` | 暖色的降饱和态（回撤柱等） |

**唯一暖色铁律**：`--btc` 只允许出现在 ₿ 符号、BTC 供应曲线、每股含币量数字、海床币阵的光——即「资产本体」。按钮、链接、hover、装饰一律冷色。访客自行得出结论：页面上唯一有温度的东西是币。

### 1.3 字体（三层，各司其职）

- **标题层**：思源宋体 `Noto Serif SC`（fonts.loli.net 镜像，`media="print" onload` 异步加载，失败回退 `"Songti SC","STSong",serif`），字重 600，标题字距 `0.06–0.12em`。宋体的书卷贵气落在冷钢底上。
- **正文层**：系统黑体栈 `-apple-system,"PingFang SC","HarmonyOS Sans SC","Microsoft YaHei",sans-serif`，17px/1.9。
- **数据层**：细等宽 `"SF Mono",Menlo,Consolas,monospace`，用于数字、深度标注、坐标；中文单位随黑体。
- ₿ 字符一律 `Georgia,serif`（徽记字形）。

### 1.4 中文为主

界面文字、章节名、图注、深度标注全部中文（数字保留阿拉伯数字，如「水深 −1025 米」）。英文仅存三处：CASPIAN VAULT 字标、页脚一行铭文、徽记缩微文字。

### 1.5 纹理与部件语言

- **等深线**：章节底纹与分隔线的母题——柔和闭合曲线（海的年轮），线宽 0.6–1px，`--steel` 低透明度。替代 v3 弦纹。
- **图录网格**：卡片用极细 hairline 边框（1px, `--steel` 30%），无圆角或 2px 微圆角；替代瓷面噪点。
- **声呐环**：点击反馈为一圈细环扩散（替代涟漪）。
- **右缘深度刻度**：全站签名部件。0 米 → −1025 米的细等宽刻度随滚动推进，中文标注。
- 底款印章、展牌（gx）、瓷器语言全部退场。

## 2 · Logo「等深归心」（B1）

- **图形**：3–4 条按里海真实测深走势的柔和闭合等深环向内收敛，环线 `--titan`，外环最淡、内环最实；最深处（南盆 −1025 米）居橙色 ₿（`Georgia`）。精工执行：曲线光滑、粗细有序，非草图。
- **派生四件套**：
  1. 方形单标（favicon/头像；缩至 16px 时减为双环+₿）；
  2. 横排组合标（图形 + `CASPIAN VAULT` 冷排字 + 小字「里海金库」），导航用；
  3. HERO 大图形（与海盆图分工：徽记用于品牌落款，不与主海图争位）；
  4. 页脚铭章（图形 + 缩微文字 `42°N 51°E · NOTHING FLOWS OUT`）。
- 替换现 v3 的「圆角方框水滴₿」（站长判死刑的那枚）。

## 3 · Slogan 与文案规则

- **中文主 slogan（HERO 铸字）**：**「聚流成海，藏金于渊。」** 思源宋 600，八字对仗，字距舒展。
- 英文 `Nothing flows out.` 从 HERO 移除，仅存于页脚铭文与徽记缩微文字（与 §1.4 的三处清单一致）。
- HERO 论纲句在 v3 基础上微调，落「存/藏/守」的意象；**合规硬约束：全站不得出现「升值/增值/回报」等暗示收益的词**（红线：不承诺回报）。「未来更值钱」的感觉只由时间与积聚意象（聚、沉、岁月）承载。
- 正文文案 v3 全保留；新增图注/标注按本节语气写（冷、雅、短）。

## 4 · HERO 规格

### 4.1 海盆图（艺术品级，替代 v3 简化轮廓）

- 按真实地理重描：宽北盆与伏尔加河口三角洲、**东侧卡拉博加兹湾**（近闭潟湖+窄海峡）、阿普歇伦半岛收腰、浑圆南盆。参照站长提供的地图截图。
- 测深分层光：北盆浅（亮青）→ 南盆深（近黑），2–3 层渐变 + 细等深线；岸线 1px `--titan`。
- 尺寸预算：HERO SVG ≤ 18KB。

### 4.2 编排（时间轴，缓动统一 `cubic-bezier(.22,.61,.2,1)`）

1. **0–1.2s 水体显形**：海盆以不透明度+柔和遮罩自暗中浮现（如月光落在水面）；此阶段**无任何线条**。
2. **1.2–2.8s 河流入海**：伏尔加、乌拉尔、库拉等 4–5 条河以「细光丝」`stroke-dashoffset` 滑入，线端一点微亮溶进水体后熄灭；**全程无箭头**；各河起始错开 0.2s，相互交叠。
3. **常驻态**：水面极缓的光泽呼吸（20s 周期，transform/opacity）；河流每 12–18s 随机一条轻闪一次。
4. **下滚=下潜**：HERO 下接通幅「海床带」：有光的 ₿ 币阵——景深、光衰、前排微亮（橙）、后排隐入黑暗，标注「教学示意 · 非实际持仓数」；背景水色随滚动渐深。
5. `prefers-reduced-motion`：直接呈现全部终帧，无动画。

### 4.3 眉标与法务行

「品牌展示 · 非要约邀请 · 筹建中」及暂定名 tag 原样保留，样式随新系统。

## 6 · 动画与性能规范

- 只用 `transform`/`opacity`；单个 IntersectionObserver 驱动全部 reveal（淡入+2px 上移，无弹跳）。
- 滚动联动（水色渐深、深度刻度）用 rAF 节流或 CSS scroll-driven（带回退）。
- 新增 JS ≤ 4KB；总页面（单文件）目标 ≤ 120KB。
- 兼容承接 v2 优化轮：iPad 768 断点体系、`overflow-x:hidden`、四边定位不写 `inset`、固定高替代 `aspect-ratio`、`renderer=webkit`、`format-detection`、safe-area。

## 7 · 不变项（红线自检清单）

- 零构建单文件、双击可开、离线可用；webfont 仅 loli.net 镜像且失败优雅回退。
- 全页零募资语言；「待律师终审」标注、免责全文、暂定名 tag 逐字保留。
- OG/meta、hreflang（无）、analytics 追踪行（`data-site="caspian"`）保留。
- 发布前跑：768/1024/390/360 视口台 + 控制台零错误 + reduced-motion 检查 + 「升值/增值/回报/保证」全文搜索为零。

## 8 · 验收标准

1. 全站找不到任何箭头符号与简笔画式示意图；
2. 海形经得起与真实地图并排对照（含卡拉博加兹湾）；
3. HERO 首帧只有水体，河流后至且为光丝；
4. 界面英文仅剩字标+页脚铭文+徽记缩微三处；
5. 「聚流成海，藏金于渊。」以思源宋铸于 HERO；
6. 暖色只在资产本体上出现；
7. 动画 60fps（transform/opacity only），reduced-motion 全静态；
8. §7 不变项逐条通过。

## 9 · 范围外

胡侃比特站不动；文案主张不重写（只做图注级新增）；analytics 后端不动；logo 商标注册事宜另行（命名备忘跟踪）。
